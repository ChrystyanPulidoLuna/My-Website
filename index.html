Projects

Raspberry Pi Security Camera

The Raspberry Pi Security Camera is a Python-based surveillance system that streams live video over a network from a Raspberry Pi using picamera2. A client device can view and record the feed using OpenCV, with optional motion detection recording for enhanced security. Users can configure frame rate and resolution to optimize performance, and the system supports cross-platform compatibility across Windows, Mac, and Linux.

Salary Calculator

The Salary Calculator is a data visualization and analysis tool that allows users to analyze salary trends by job title, education level, experience, age, and gender. The tool can sort data into categories, generate graphs, and function as a salary estimator by predicting salaries based on trends from the data. The program reads a CSV file and uses salary data provided by Kaggle.
